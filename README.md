## Instructions

### Run with Docker

Be navigated to this root directory in a terminal and enter command:

`docker-compose up`

When finished, the web client will be available on port 3000

http://localhost:3000/

### Tear down

`docker-compose down`

### Rebuild

`docker-compose up --build`
